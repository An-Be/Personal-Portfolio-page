Moviefy

Moviefy is an ASP.NET Core MVC App that uses CRUD operations and policy-base authentication. Anon users can see the list of movies available, regular users without a policy can edit, add, list, and watch movies. Users with policy, can do everything including delete movies.

Source Code

Non-Traditionals Blog

Non-Traditionals blog is an ASP.NET Core MVC app that allows authenticated users to post, edit, view, and delete blogs. They can only edit and delete blogs they themselves have created. Anon users can view blogs but can not make any changes until they have registered.
